Choose your Paths

Structured internships: learning sprint + build sprint = job-ready skills

DevOps

Learn:

Get hands-on with the tools of the trade, mastering Linux, Git, CI/CD, Docker, and monitoring & logging.

Build:

Showcase your expertise by creating a full CI/CD pipeline, a production-ready containerized application, and a robust multi-container app.

Cloud Computing

Learn:

Master the essentials of cloud computing, from core concepts and IaaS vs PaaS to managing compute, storage, networking, IAM, and cost basics.

Build:

Showcase your skills by deploying a web app, setting up storage and a CDN, and automating infrastructure with basic infrastructure-as-code.

Generative AI

Learn:

Master the fundamentals of Generative AI, from understanding how models generate content to advanced prompt engineering techniques and ethical considerations.

Build:

Showcase your expertise by building a prompt playbook for various tasks, creating mini-agents to automate workflows, and developing domain-specific prompt systems.

Data Science

Learn:

Learn to speak the language of data. You'll master Python for DS, data wrangling, and exploratory data analysis to build powerful machine learning models and tell compelling data stories.

Build:

Showcase your skills by completing an end-to-end machine learning project and building a professional portfolio of interactive dashboards and notebooks.

Each program includes mentor check-ins, code reviews, and a final showcase.
